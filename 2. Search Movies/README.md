## Run

type this command to run:

```Shell
npm start
```

search movie:

```Shell
http://localhost:3000/movies?q={your_query}&page={page default 1}
```

detail movie:

```Shell
http://localhost:3000/movies/{imdbID get from search}
```

## Test

type this command to test:

```Shell
npm test
```
